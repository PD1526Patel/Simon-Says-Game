# 🎮 Simon Says Game

A simple and interactive **Simon Says Game** built using web technologies. The game challenges your memory by generating a sequence of colors that you must repeat correctly. Each round increases the difficulty!

---

## 🧠 How It Works

- The game shows a sequence of colors (with animations/sounds)
- You must repeat the same sequence by clicking buttons
- Each correct round adds a new step to the sequence
- The game ends if you make a mistake

---

## 🚀 Features

- 🎨 Interactive UI with color buttons
- 🔊 Sound effects for each button
- 📈 Increasing difficulty levels
- 💡 Visual feedback for correct/incorrect inputs
- 🔄 Restart option after game over

## ▶️ How to Play

1. Start the game (press a key / click start button)
2. Watch the sequence carefully
3. Repeat the sequence by clicking the buttons
4. Progress to higher levels
5. Try to beat your high score!
